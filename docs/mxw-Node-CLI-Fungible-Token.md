# Fungible Token Module


### How to use
Fungible Token Consist of Msgs:   

```
token-createFungibleToken 
- This is the message type used to create the fungible token. 

token-setFungibleTokenStatus 
- This is the message type used to set the status of a fungible token. 

token-transferFungibleToken 
- This is the message type used to transfer the item of fungible token. 

token-mintFungibleToken 
- This is the message type used to mint the item of fungible token. 

token-burnFungibleToken 
- This is the message type used to burn the item of fungible token. 

token-transferFungibleTokenOwnership 
- This is the message type used to transfer the ownership of a fungible token. 

token-acceptFungibleTokenOwnership 
- This is the message type used to accept the ownership of a fungible token. 


token-setFungibleTokenAccountStatus 
- This is the message type used to set the account status of a fungible token.
```
